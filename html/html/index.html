My Favorite Learning Applications



SoloLearn

SoloLearn is an application that was founded in 2013 by Yeva Hyusyan and David Kocharyan. It is an application that was developed to help people learn to code or program. This application has many different languages for the user to study or learn; some of the programming languages include:

  • Java
  • JavaScript
  • HTML
  • C++
  • Python
I personally have used this app for about a year now and it has made learning coding languages MUCH easier to comprehend. I could not recommend this app enough for anyone who wants an advantage when learning to code. The best part of the app is that it is free! It is also in a fill-in-the-blank format and also features a 'code playground' for people who would like to show off what they have learned to other SoloLearn users. It is set up similar to a social media site like FaceBook or Twitter to where other users can follow you and your progress to learning a language.

How To Get Started

To download the app on your phone, go to the app store for your phone, search 'SoloLearn' and it will most likely be the first result in your search. Download the app and then you will be prompted to make an account(don't worry, it is free to sign up). You can then select your language that you wish to study and then cruise your way through the course on that language. Each language course is broken down into different sections that focus on different parts of the language such as variable types, loops, classes, etc. Below is a button that will take you to the homepage of SoloLearn.
